# PD_Tool
A simple tool for working with Project Diva DT/FT/F/F2/X files

# Dependencies:
+ `.NET Framework 4.6.1`: Required to run/build PD_Tool C# project.

# Tools:
- `FARC ExtPack`
- `DIVAFILE EncDec`
- `DB_Tools`
  - `Auth DB Converter`
  - `Bone DB Converter`
- `Converting Tools`
  - `A3DA Converter`
  - `STR Converter`
